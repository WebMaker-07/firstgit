Configuring Git

  1. Go to VS code and click terminal --> New Terminal
  2. Check if git is already installed by typing git --version in the command line
  3. Configure git in your local computer
    • Set your username:
      git config --global user.name "your name"
    • Set your email address :
      git config --global user.email "your email address"
  4. To initialize the project directory type git init

Git add and coomit

  1. To see the status of the file in you directory type git status
  2. to track new file use the command git add:
    Syntax: git add .(to add groupu of file)
    git add filename(for individual file)
